Class RouterBuilder

General router builder. Creates routes on top of passed Express router based on data about desired routes - their type, name (base url) and schema of the data to be returned

For GeoJSON data, automatically creates / and /:id routes, adds filtering parameters (geo, city district, limit, offset, last updated)
